Description of the important tags

1.Heading Tag
Heading tags are most important tags in html. H1 tag is used for the most important headings and rest of the heading tags for other type of headings. They are numbered according to their sizes.
2.Emphasis Tag
Emphasis tag is used to emphasize any text which is more important than rest of the text. For example: the word Rainbow is emphasized here.
3.Paragraph Tag
Paragraph tag is important in html as it is used for every paragragh. If we want to change or add a new paragraph, we need to use this tag.
4.Superscript Tag
If we want to add any number with Superscript on it, we need to use the Superscript tag. For example: 11th.
5.Important text tag
For anything which we want to highlight from rest of the text and make it bold we use strong tag for it. For example: the word Name is highlighted here as an important text.
